CTable JavaScript CRUD Framework

CTable is a JS library to make CRUD (CReate, Update, Delete) web tables.

Key features

  • Modern browsers support (ES6 Javascript with jQuery)
  • Asynchronous (AJAX based)
  • Server technology agnostic
  • Server-side sorting and pagination
  • Sub-table support
  • Easily extends by column class inheritance
  • Good looking and mobile-ready with Bulma CSS and Font Awesome icons

Standart column classes

  • Text field CTextDataColumn
  • Combobox CSelectColumn
  • Control buttons CCommandColumn

Helper classes

  • Pagination CPagination
  • Div based record editor CDivEditorRecord

Usage exapmle

1. Add to page header:

            <script type="text/javascript" src="query/jquery.min.js"></script>
            <script type="text/javascript" src="ctable/ctable.js"></script>
            <link rel="stylesheet" href="css/bulma.min.css">
            <link rel="stylesheet" href="fa/css/all.min.css">

2. After documet ready:

            var t = new CTable({endpoint: "/demo.php"});
            t.set_record_class(CDivEditorRecord);
            t.add_column_class(CSubtableColumn,{title:'Exp', table: function(parent_record){
            
                var tt = new CTable({endpoint: "/demo.php"});
            
                tt.set_record_class(CDivEditorRecord);
                tt.add_column_class(CColumn,{column:'id', title:'ID', visible_editor:false, visible_column:false});
                tt.add_column_class(CTextDataColumn,{column:'firstname', title:'Имя', validate:"\\S+"});
                tt.add_column_class(CTextDataColumn,{column:'lastname', title:'Фамилия', validate:"\\S+"});
                
                return tt;
            }});
            
            t.add_column_class(CColumn,{column:'id',
                                        title:'ID',
                                        visible_editor:false,
                                        visible_column:false});
            
            t.add_column_class(CTextDataColumn,{column:'firstname',
                                                title:'Firstname',
                                                validate:"\\S+"});
            
            t.add_column_class(CTextDataColumn,{column:'lastname',
                                                title:'Lastname',
                                                validate:"\\S+"});
            
            t.add_column_class(CSelectColumn,{column:'sex',
                                              title:'Sex',
                                              options:[['m','М'], ['f':'F']],
                                              validate: true});
            
            t.add_column_class(CCommandColumn,{});
            
            t.set_pagination_class(CPagination,{});
            
            t.build_table($('#mainbox'));
            
            t.select();
